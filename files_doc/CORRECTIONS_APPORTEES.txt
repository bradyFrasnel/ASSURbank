   - rôle = 'ROLE_BANQUE'

   Table CLIENT:
   - id (INT, PK)

   - téléphone, rôle (ROLE_CLIENT ou ROLE_ADMIN)

   - id (INT, PK)
   - numéro_compte, type, solde, devise
   - statut, date_création
   - client_id (FK → Client)

   Table TRANSACTION:
   - id (INT, PK)
   - montant, type, libellé, frais
   - date_transaction
   - compte_source_id, compte_destination_id (FK)
   - statut (succès/échoué)
